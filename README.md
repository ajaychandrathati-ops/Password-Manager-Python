## Project Description

Secure Password Manager is a Python-based desktop application that allows users to securely store and manage their passwords. The application uses encryption techniques to protect sensitive information and provides an easy-to-use graphical interface.

## Features

* Password Encryption
* SQLite Database Storage
* Tkinter GUI
* Add Passwords
* Delete Passwords
* Secure Credential Management

## Technologies Used

* Python
* Tkinter
* SQLite
* Cryptography (Fernet Encryption)

## How to Run

1. Install the required package:

```bash
pip install cryptography
```

2. Run the application:

```bash
python password_manager.py
```
